t-sne-lab is a laboratory for you to interactively explore a high-dimensional numerical dataset using the t-sne visualization. It reads a csv file and allows you to choose which categorical field to be used for color painting, and which categorical field to be used for labeling the data points. You can also try different values of the perplexity and epsilon parameters.
